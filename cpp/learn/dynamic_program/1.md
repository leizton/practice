# 《两个序列的二维dp》
dp[i,j]: 序列p[0:i]和序列s[0:j]的最优值。 如果考虑空字符串则是p[0:i-1]和s[0:j-1]
~~~c++
// 两种模式都是依赖上、左、斜上方三个值
// 向前推导
for i : [1,m]
  for j : [1,n]
    // 网格里由上、左、斜上方推导dp[i,j]
    dp[i,j] = { dp[i-1,j-1] dp[i-1,j] dp[i,j-1] }
// 向后寻找
for i : [1,m]
  for j : [1,n]
    if p[i] == s[j]
      dp[i,j] = { dp[i-1,j-1] dp[i-1,j] dp[i,j-1] }
    else
      dp[i,j] = { dp[i-1,j-1] dp[i-1,j] dp[i,j-1] }
~~~
