## 给定一个字符串 s
~~~c++
for (int i = 1; i < n; i++)
  for (int j = i; j < n; j++)
    if s[i] == s[j]
      dp[i][j] = ...;
    else
      dp[i][j] = ...;
~~~

# Distinct Ways

# Merging Intervals
递归式： dp[i][j] = dp[i][k] + result[k] + dp[k+1][j]
