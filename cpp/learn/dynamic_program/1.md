# Minimum (Maximum) Path to Reach a Target
递归式：f(i) = min{f(i-1), f(i-2), ...}

# Distinct Ways
递归式：f(i) = f(i-1) + f(i-2) + ...

# Merging Intervals
递归式： dp[i][j] = dp[i][k] + result[k] + dp[k+1][j]
